เพิ่มพนักงานใหม่
1️⃣
ไปที่ Admin → จัดการผู้ใช้ → กด "+ เพิ่มผู้ใช้"
2️⃣
กรอก Username และชื่อ-นามสกุล

3️⃣
เลือกสิทธิ์: User หรือ Admin
User = พนักงานทั่วไป | Admin = ผู้ดูแลระบบ
4️⃣
แจ้ง Username ให้พนักงาน
พนักงานจะต้องไปตั้งรหัสผ่านเองผ่านหน้า "ตั้งรหัสผ่านครั้งแรก" — Admin ไม่รู้รหัสผ่านของพนักงาน

FAQ
คำถามคำตอบ
Login ไม่ได้ บอกว่า "ไม่พบชื่อผู้ใช้" ติดต่อ Admin ให้สร้างบัญชีให้ก่อน
Login ไม่ได้ บอกว่า "รหัสผ่านไม่ถูกต้อง" ตรวจสอบ Caps Lock หรือให้ Admin รีเซ็ตรหัสผ่านให้
วางแผนแล้ว แต่เพื่อนไม่เห็น ต้องกด "บันทึกแผน" ก่อน และให้เพื่อนรีเฟรชหน้าเว็บ
วางแผนย้อนหลังได้ไหม? ได้ครับ กดลูกศร ← ในหน้า "วางแผนของฉัน" เพื่อย้อนกลับสัปดาห์ที่แล้ว
จะใช้จากมือถือได้ไหม? ได้เลย เปิด URL ผ่าน Safari หรือ Chrome บนมือถือ แนะนำให้ Add to Home Screen
ข้อมูลหายหลังปิด browser? ข้อมูลเก็บใน Cloud ไม่หาย ยกเว้น Session Login จะหมดเมื่อปิด browser ต้อง Login ใหม่
อยากเพิ่มสถานที่ทำงานใหม่ ติดต่อ Admin → เมนู "สถานที่" → เพิ่มสถานที่ใหม่ได้
วันหยุดนักขัตฤกษ์ไม่แสดง ติดต่อ Admin → เมนู "วันหยุด" → เพิ่มวันหยุดนั้น
